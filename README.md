# Java OOP Payment Gateway System

## Overview
This project demonstrates abstraction and factory design pattern in Java using a payment system.

## Features
- Abstraction using abstract class
- Polymorphism with method overriding
- Factory Design Pattern
- Multiple payment modes:
  - UPI
  - Netbanking
  - Credit Card

## Tech Used
- Java
- OOP Concepts

## How It Works
Based on user input, the factory class returns the appropriate payment object and executes the payment method.

## Example
Input: UPI  
Output: Payment processed using UPI
